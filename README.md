# Chatbot Ley 7/2022 - Residuos y Suelos Contaminados

## Objectives
- Provide users with information about waste and contaminated soils.
- Allow users to ask queries and receive accurate answers based on legal frameworks.

## Features
- **User-friendly Interface**: Simple and intuitive interface for easy interaction.
- **Natural Language Processing**: Understands user queries in natural language.
- **Database Integration**: Accesses a wide range of information on legal requirements.

## Technology Stack
- **Frontend**: React, Bootstrap
- **Backend**: Node.js, Express
- **Database**: MongoDB

## Installation Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/davidsfambiental26/chatbot-ley6-2022.git
   ```
2. Navigate to the project directory:
   ```bash
   cd chatbot-ley6-2022
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the application:
   ```bash
   npm start
   ```

## Usage Instructions
### Frontend
- Navigate to `http://localhost:3000` in your browser to access the chatbot.

### Backend
- The backend runs on `http://localhost:5000`. Ensure the server is started in the project folder before using the frontend.

## Project Structure
- `frontend/`: Contains all frontend code.
- `backend/`: Contains all backend code.
- `docs/`: Documentation files.
- `tests/`: Unit tests for both frontend and backend.

## Testing Guide
- To run tests, use the following command in both the frontend and backend directories:
  ```bash
  npm test
  ```
